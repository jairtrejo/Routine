Routine

Routine was developed during the week from April 10 to April 17 as an entry for Indextank and Factual's contest. The task was "hacking up an application that uses both IndexTank real-time custom search API and Factualâ€™s dataset of 14M US businesses."

Building Routine in such a short timespan was very interesting. It is impressive to see how much one can accomplish in one week using readily available tools; as a developer it is quite nice to be able to rely in such tools and concentrate in the application itself. In this page I would like to give an overview on the technologies and resources involved in creating this app.

For UI presentation I used the Atatonic CSS Framework, which made it very easy to work with a grid and have a nice looking page from the start. I changed the default font size and typeface because I prefer big letters and a sans-serif typeface for web apps.

For UI behaviour I used JQuery, which has been my javascript library of choice since I started on web development. The side-scrolling interface was made with JQuery Tools and for the facebookish window that shows the itinerary's detail I used Facebox.

On the backend I used CherryPy as web framework. It really only performs two tasks: searching for businesses and sorting them in an efficient route.

The search is performed by Indextank. The service is absolutely great, and I got the hang of it quite quickly. At first it striked me as a little slow, but when I deployed the app in my server I realized it was my ugly internet service. Now it is truly lightning fast! Besides, it was very easy to use and very well documented. I wholehertedly recommend it.

The businesses database came from Factual. It is quite comprehensive and as it comes as a simple csv file, it was very easy to upload to Indextank with a simple script. Nonetheless, it would be cool if we could just upload the csv to Indextank and get it indexed automatically.

All the maps are powered by Google Maps' Javascript API. It was easier to use than I expected, and the geocoding service works like a charm.

Finally, I'd like to thank interactivemania, from where I took the little cross icon and WPZOOM from where I took the magnifying glass.
